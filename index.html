👨‍👩‍👧 Deling og synk
Koble til en delt liste så du og andre (f.eks. kona) ser og endrer de samme oppgavene i sanntid.
Ikke tilkoblet
Slik setter du opp gratis synk (én gang)
  1. Gå til console.firebase.google.com og opprett et gratis prosjekt.
  2. Åpne Build → Realtime Database → Create database (velg en region, f.eks. europe-west1).
  3. Under Rules, tillat lesing/skriving på husholdninger:
    { "rules": { "households": { "$h": { ".read": true, ".write": true } } } }
  4. Kopier database-URL-en (ser ut som https://…firebasedatabase.app) inn i feltet over.
  5. Velg en hemmelig husholdnings-kode og trykk Koble til. Del lenken med kona – hun trykker bare «Koble til».
Merk: hvem som helst med URL-en og koden kan se/endre listen. Bruk en kode som er vanskelig å gjette.
